# 🏏 MoneyBall: A Machine Learning-Based Cricket Player Similarity Engine

A simple ML-powered system that finds cricket players similar to a given player based on performance metrics like average runs, strike rate, consistency, and boundary percentage.

## 🚀 Features

* Player similarity using cosine similarity
* Feature engineering from raw stats
* Scalable and easy to extend
* Optional Streamlit UI

## 🧠 Tech Stack

Python, Pandas, Scikit-learn, NumPy, Streamlit

## ▶️ Usage

```bash
pip install pandas scikit-learn numpy streamlit
python main.py
```

## 📌 Example

Input: `Virat Kohli`
Output: Top 5 similar players with similarity scores
